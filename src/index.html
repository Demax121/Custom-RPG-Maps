HOW IT WORKS

What the icons on the sidebar mean

The globe icon opens a tab where markers that are either fetched from a database or a locally stored JSON file are displayed in the form of lists grouped into overlays.
When you click on the name of the overlay, the list containing markers will appear. Clicking on the marker name zooms the map onto it.



The marker icon opens the custom markers tab. These markers are created by the user and are not stored anywhere, not even in local or session storage. When you create new markers, they are separated into corresponding overlays in a similar way as in the tab with markers fetched from the database. These markers can then be exported either in JSON or TXT format, already formatted for the SQL INSERT statement. The delete button removes the created marker, and if the overlay list doesnâ€™t contain any markers, it is removed from the tab. The rename button allows you to rename the marker if you decide to do so.
